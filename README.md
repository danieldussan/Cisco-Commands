# Cisco Commands — Organized Reference

Repositorio con comandos, ejemplos y ejercicios para aprender y practicar administración de redes Cisco (enfoque CCNA).

Contenido principal:

- `01_comandos_basicos.md` — comandos básicos y explicaciones de argumentos
- `02_vlan_y_trunking.md` — VLANs, trunks, voice VLAN y router-on-a-stick
- `03_routing_estatico_y_dinamico.md` — rutas estáticas y conceptos de routing
- `04_subnetting_y_vlan_interrouting.md` — subnetting, VLSM y ejemplos de subinterfaces
- `05_dhcp_nat_dns.md` — DHCP, NAT (dinámico y PAT) y DNS básicos
- `06_ospf_eigrp_rip.md` — OSPF, EIGRP, RIP con ejemplos y verificación
- `07_acl_y_ace.md` — ACLs estándar y extendidas, con ejemplos y verificación
- `08_seguridad_basica.md` — seguridad de acceso, port-security, hardening básico
- `09_ipv6_configuracion.md` — comandos y verificación IPv6
- `10_comandos_utiles_y_troubleshooting.md` — troubleshooting y comandos útiles
- `cheatsheet.md` — hoja rápida de comandos para consulta inmediata
